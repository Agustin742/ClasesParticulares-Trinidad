# Como iniciar un proyecto de TS

## 1. Instalar TypeScript y tsx

```bash
npm install -D typescript tsx
```

## 2. Crear un archivo
```ts
// index.ts
const mensaje: string = "Hola TypeScript";
console.log(mensaje);
```

## 3. Ejecutarlo

```bash
npx tsx index.ts
```

## 4. Agregar scripts al package.json

```json
{
  "scripts": {
    "dev": "tsx index.ts"
  }
}
```

## 5. Inicializar TypeScript correctamente
Generá el tsconfig.json

```bash
npx tsc --init
```

## 6. Correr proyecto:

```bash
npm run dev
```

## 7. Correcion para la exportacion de clases

En tsconfig.json
dejar el modulo en nodenext

```json
"module": "nodenext",
```

### Agregar type a Package

En el package.json agregar type, debe quedar de la siguiente manera todo el archivo

```json
{
  "type": "module",
  "scripts": {
    "dev": "tsx index.ts"
  },
  "devDependencies": {
    "tsx": "^4.21.0",
    "typescript": "^6.0.3"
  }
}
```
